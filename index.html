Teaching: Prompting Activities

Click a button to reveal the student prompt.

Zero-shot Prompt

Create a zero-shot prompt based on a Wikipedia page or news article. Output (when used) should be a bullet list.

Student Prompt

Summarize the key geographic, political, and environmental aspects of [topic] in 6–8 bullet points. Include dates, places, impacts, and recovery efforts.

Few-shot Prompt

Use short examples to steer structure/style. Output target: 50–75 words.

Student Prompt

Example 1: [paste short example]
Example 2: [paste short example]
Task: In 50–75 words, explain the significance of [topic] with attention to place, people, and policy.

Instructional — Journalistic

Output target: 500–750 words.

Student Prompt

Write a 500–750 word article on [topic]. Use clear, engaging language, include key places/times, and cite credible sources for follow-up reading.

Instructional — Industry Report

Output target: 500–750 words.

Student Prompt

Draft a 500–750 word industry brief on [topic]. Include an executive summary, key metrics, stakeholders, and a short risk section.

Contextual — Novice Audience

Output target: 250–350 words.

Student Prompt

Explain [topic] for readers new to the subject. Use everyday metaphors, a simple map description, and gently paced structure (250–350 words).

Contextual — Expert Audience

Output target: 250–350 words.

Student Prompt

Summarize current research on [topic], citing datasets, uncertainties, and recent papers (250–350 words).

Role-playing — Citizen

Output target: 350–500 words.

Student Prompt

Write 350–500 words from the perspective of a local resident responding to [event]. Include place-based details, concerns, and community values.

Role-playing — Economic Stakeholder

Output target: 350–500 words.

Student Prompt

Write 350–500 words as an industry stakeholder discussing [event] impacts, citing economic indicators and land-use policy considerations.

Ideal Structure Prompt

Output target: 750–1000 words.

Student Prompt

Using [topic] as a case, produce a 750–1000 word developed report with:
• Title & 2-sentence abstract
• Geographic, political, environmental context
• Timeline of key events
• Stakeholder analysis
• Risks, uncertainties, and sources
• 3 next-step questions for deeper inquiry

About

I'm a geographer focused on remote sensing, GIS, and clear, compelling map design. My work spans land surface temperature analysis, change detection, and public-facing data visualization for teaching and research.

Tools I use include ArcGIS Pro, QGIS, GDAL, Python (rasterio, geopandas), and web mapping stacks (Leaflet, Mapbox GL). I teach students how to turn complex spatial data into stories that inform decisions.
